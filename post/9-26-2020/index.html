如何在Mac下使用Homebrew来安装运行MongoDB

步骤0. 事先准备

安装XCode

Apple的XCode包含了使用brew所必须的命令行工具，可在App Store免费下载

安装Homebrew

Mac OS系统并没有默认提供Homebrew, 我们可在Homebrew官方指导获取下载帮助。或是在终端运行如下指令：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

步骤1. 追踪MongoDB的官方brew仓库

在终端运行如下指令追踪获取MongoDB的官方brew仓库

brew tap mongodb/brew

步骤2. 安装

@后的数字为最新版本号。在2020年9月，最新的版本号为4.4

brew install mongodb-community@4.4

步骤3. 运行和停止运行

运行

brew services start mongodb-community@4.4

停止运行

brew services stop mongodb-community@4.4

步骤4. 检查MongoDB是否在运行

ps aux | grep -v grep | grep mongod

步骤5. 连接并使用MongoDB

mongo
